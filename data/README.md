# Data

- **\[Dataset\]**: The dataset that I am pulling from is the TidyTuesday "Pokemon" set. 
This set contains all Pokemon from the original 151 Generation 1 group up to Generation 7 which now includes over 800 Pokemon! 
The data set includes an array of their battling information like types (type_1 and type_2), attack/defense, special attack/defense, 
health points (hp), speed and so on.

# Codebook for Pokemon Dataset

## Variable Names and Descriptions:

- **id: The unique ID of each Pokemon.**
- **pokmemon: The name of each pokemon.**
- **type_1: The primary type.**
- **type_2: The secondary type.**
- **hp: The HP (hit points).**
- **attack: The attack points.**
- **defense: The defense points.**
- **special_attack: The special attack points.**
- **special_defense: The special defense points.**
- **speed: The speed.**
- **generation_id: The generation ID of each Pokemon.**
- **starter: If the Pokemon is a starter or not.**
- **evo_stage: Which evolutionary stage is the given Pokemon.**
- **region: List of region(s) where the given Pokemon is found in the regional Pokedex.**
- **base_stat_total: Sum of the hp, attack, defense, speed, special_attack, and special_defense variables**

## Data Types:

- **id: integer**
- **pokemon: character**

- **type_2: character**
- **hp: integer**
- **attack: integer**
- **defense: integer**

- **speed: integer**
